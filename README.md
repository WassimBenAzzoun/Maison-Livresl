# 📚 Gestionnaire de Bibliothèque  
Projet PHP / JavaScript

## 👥 Membres du projet
- **Wassim Ben Azzoun**  
- **Rayen Selmi**

## 🏫 Réalisé à
**Tekup University**

## 📝 Description du projet
Ce projet consiste à développer une application web permettant la gestion complète d’une bibliothèque.  
Elle offre plusieurs fonctionnalités essentielles pour l’administration et le suivi des livres, des utilisateurs et des emprunts.

## 🚀 Technologies utilisées
- **PHP** (Back‑end)
- **JavaScript** (Front‑end)
- **MySQL** (Base de données)
- **HTML/CSS** (Interface utilisateur)

## 📌 Fonctionnalités principales
- Gestion des livres (ajout, modification, suppression)
- Gestion des utilisateurs
- Gestion des emprunts et retours
- Recherche avancée
- Interface simple et intuitive

## 🏁 Objectif
Fournir un outil efficace et moderne pour faciliter la gestion d’une bibliothèque universitaire ou personnelle.
